Este archivo puede ser ejecutado desde la línea de comandos de windows siguiendo las siguientes instrucciones:
1.- Ejecutar el cmd
2.-Ingresar a la ruta donde esta guardado en archivo p.ej. " cd C:\Users\Vivian\Desktop>"
3.- Identificar la ruta en la que se encuentra el compilador de java "javac.exe" p.ej. "C:\Program Files\Java\jdk1.8.0_161\bin\javac.exe"
4.- Copiar la ruta anterior en cmd seguido del nombre del archivo Cuadrado.java p.ej. "C:\Program Files\Java\jdk1.8.0_161\bin\javac.exe" Cuadrado.java
5.Copiar la ruta del archivo en cmd seguido del nombre del archivo Cubo.java p.ej. "C:\Program Files\Java\jdk1.8.0_161\bin\javac.exe" Cubo.java
6.-Copiar la ruta anterior en cmd seguido del archivo Objetos.java p.ej. "C:\Program Files\Java\jdk1.8.0_161\bin\javac.exe" Objetos.java
7.- Escribir 'java' seguido del nombre del archivo 'Objetos', respetando mayusculas y omitiendo la extensión p.ej. java Objetos
8.- El programa empezará a ejecutarse, pedirá que se ingrese un numero ENTERO para Asignar el valor al lado del objeto
9.- Una vez ingresado el número se presiona el boton ENTER y el programa nos mostrará una lista de opciones para visualizar opciones de cálculo disponibles.
10.-Se ingresa el número correspondiente a la opcion que se quiere visualizar
11.- Una vez ingresado el número se presiona el boton ENTER y el programa nos mostrará el resultado de la operación elegida.
12.Para salir del programa se selecciona la opcion 6 "Salir" y se presiona ENTER.
